# Learning Design Patterns

Repositorio para combinar aprendizaje teórico con ejercicios prácticos sobre patrones de diseño.

Estructura propuesta:

- `docs/` : Contenidos teóricos (introducciones, explicaciones por categoría).
- `exercises/` : Ejercicios y enunciados para practicar.
- `solutions/` : Soluciones referenciadas separadas (opcional).
- `templates/` : Plantillas para crear nuevas lecciones y ejercicios.
